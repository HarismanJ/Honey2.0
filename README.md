## Installation

### Prerequisites
1. Node.js and npm installed on your system.
2. Google Chrome browser.
   
### Steps
1. Clone this repository:
   
   git clone <repository-url>
   cd <repository-folder>

2. Install dependencies:
   
   npm install

3. Start the backend server:

   node server.js
